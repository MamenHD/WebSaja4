Program

Beragam Program

Di jurusan Rekayasa Perangkat Lunak (RPL), siswa mengikuti berbagai program praktis untuk mengembangkan keterampilan teknologi. Beberapa program yang dilakukan di antaranya adalah pembuatan website, pengembangan aplikasi mobile, pelatihan teknisi HP, dan promotor teknologi. Pembuatan website dan aplikasi bertujuan untuk melatih siswa dalam perancangan, coding, dan pengujian produk digital. Pelatihan teknisi HP membantu siswa memahami perbaikan perangkat keras dan perangkat lunak pada ponsel. Selain itu, program promotor memberi kesempatan bagi siswa untuk belajar cara memasarkan teknologi dan produk digital secara efektif.
